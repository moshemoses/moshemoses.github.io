About Me

Two and a half years ago I was a middle school science teacher seeking a way to bring the science of electricity to life through robotics. While exploring different teaching options I stumbled across the Raspberry Pi in all its tasty glory. While I did not end up using this platform in the classroom, I began a profound journey of tinkering and programming. My internet of things projects included a smart doorbell system and a api-monitoring hockey goal alarm. Exploring programming have lead me away from classroom teaching and toward the magic of computers and coding them. When I'm not coding I'm normally hiking with my wife and son, or doing some sort of DIY construction on my home. My retirement dreams involve building log cabins and tree-houses on my fantasy farmstead.

My Projects

Save For Later
Save For Later is a system that allows users to save web articles for later offline-reading on a mobile app. We used a Google Chrome Extension to allow users to select the articles for offline reading. The extension also scrapes articles for the plain text of the article, thereby softening the storage load requirements. The text data is then pushed to cloud storage via Node/Express server on Heroku. The data itself is actually stored on a MongoDB database which serves as our central source of truth. Our React-Native app allows users to download their saved articles to offline-storage thereby allowing them to read despite not having access to the web.

Check out our project on Git Hub

English Gematria Calculator
English Gematria Calculator is a tool for numerology enthusiasts who wish to generate the gematria (numerological value) of English words as well as find words with similar equivalence. The site utilizes a stack of Node, Express, Sqlite and plain old fashion Java Script to parse and return a string's numerological value. It then also queries and returns from a sqlite database for all English words with the same numerological value. Click here to see my code on Git Hub

Time Til Shabbat
This project, available here, utilizes Python along with a Flask backend framework to provide the start time for the Jewish Sabbath and holidays. Using your IP address to grab your relative location, it then parses the HebCal.com API to determine the start time for Shabbat. The website also utilizes a front-end Java Script countdown timer which tells users how much time remains until the Shabbat begins. Code available on GitHub
